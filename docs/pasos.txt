Pasos instalar angular

Tener node js 20+

Instalar angular 21 de forma global

npm install -g @angular/cli@21

Validar version

ng version

instala el nuevo gestor de paquetes de forma global usando el npm tradicional.

npm install -g pnpm

Conectar Angular con pnpm

ng config -g cli.packageManager pnpm

Si ya configuraste pnpm de forma global puedes instalar nuevas caracteristicas o versions:

pnpm add -g @angular/cli@21

con npnm tenes una una version instalada y ahi toma siempre nuestro commando ng new cuando creamos un nuevo Proyecto, queda unificado, ademas con add Podemos agregar nuevas librerias de forma global.

agregar Nueva libreria

pnpm add bootstrap

crear proyecto
ng new mi-proyecto-angular

crear sin instalar dependencias

ng new mi-proyecto-angular --skip-install

En lugar del npm install tradicional, pueden entrar a su carpeta y usar:

pnpm install --offline

Actualizar version de Proyecto:

ng update @angular/core@^21 @angular/cli@^21
